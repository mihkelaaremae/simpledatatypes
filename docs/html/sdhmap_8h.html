◆ sdhmap_capacity

#define sdhmap_capacity (   map)

Retrieve the amount of elements the map can store currently.

Average time complexity - O(1)
When map is a heap-type map then the capacity will be grown automatically to make space for more elements.
For stack-type maps the capacity is fixed and cannot be changed. Attempting to do so by adding more elements will trip an assert.

Parameters
[in]mapMap object to retrieve the capacity from.
Returns
Capacity of map, counted in elements (sdhmap_index_t).

◆ sdhmap_new

#define sdhmap_new (   ...)

Construct a new map.

Average time complexity - O(capacity)
If map contains a previously used map then it should be freed with sdhmap_delete.

Parameters
[in]mapMap to initialize
[in]hash_func(OPTIONAL) Hash function with the prototype sdhmap_index_t (const void *). Defaults to a library-generated function if key type is a basic type or char * (for strings).
[in]eq_func(OPTIONAL) Equality function with the prototype int (const void *, const void *). This function should return 0 for equal elements and nonzero for elements that are not equal. Defaults to NULL which means that memcmp is used to test equality instead.
[in]capacity(OPTIONAL, OMITTED) amount of elements to reserve space for. Defaults to SDHMAP_DEFAULT_CAPACITY. This option is omitted for stack-type maps.
